Grouping things

I have put this particular paragraph in a box. <div> is a box that you can put other things into. It comes in useful when you want to group things together.

You can use classes to annotate the elements in code. They can help you distinguish between the same elements that you might use for different things. Classes are not visible, but later you will learn how to use them to change appearance of things on the page.

You can have many elements with the same class.

You can also use id for a similar purpose: use it when you want to mark out a specific element

You can only have one element with a specific id though, but as a bonus you can link to that element within the page.
